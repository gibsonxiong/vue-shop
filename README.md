<h2 align="center">vue-shop</h2>
<p align="center"><b>电商项目</b></p>


## 特性

- 基于vue、vue-router、axios的电商项目，需配合koa-shop后台使用。


## 快速入门

### 安装依赖

```bash
$ npm install 
$ npm run start  #开发模式
$ npm run build:prod  #打包（生产环境）
```

### 目录说明

```
├─ config
├─ public
├─ src
│  ├─ assets
|  ├─ common      //项目公用函数
│  ├─ components  //组件文件
│  ├─ config       //配置文件
│  ├─ css          //样式
│  ├─ filters      //vue过滤器
│  ├─ js          //第三方js
│  ├─ router          //路由
│  ├─ routerCache     //基于vue-router的缓存mixin
│  ├─ services          //请求方法api
│  ├─ store          //暂时没有用
│  ├─ utils          //工具函数
│  ├─ validate          //数据校验
│  └─ views          //页面文件

```
